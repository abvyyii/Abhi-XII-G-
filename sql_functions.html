CREATE

This command is used to create a new table, database, index, or view in a database.

DROP

This command is used to delete a table, database, or view from the database

ALTER

This command is used to modify an existing database object, such as a table

ADD

This is often used with the ALTER command to add a new column to an existing table.

UPDATE

This command is used to modify existing records in a table.

SET

This command is used to assign a value to a variable or update column values in a table.

INSERT

This command is used to add new records into a table.

SELECT

This command is used to query data from a database.
